 # 📝 To-Do List

Projeto web de lista de tarefas desenvolvido em React, permitindo que o usuário adicione, visualize, marque como concluída e remova tarefas de forma simples e dinâmica.

# 🧠 Sobre o Projeto

O projeto To-Do List foi desenvolvido como parte de estudo e prática em React, focando em desenvolvimento de aplicações web interativas e responsivas.

Ele permite:

Adicionar novas tarefas
Listar tarefas dinamicamente
Marcar e desmarcar tarefas como concluídas
Excluir tarefas individualmente ou todas de uma vez
Contar tarefas concluídas

O layout é responsivo e pensado para desktop e mobile, usando componentes reutilizáveis e código organizado.

# 🛠️Tecnologias Utilizadas
React 19.2.0 ,
JavaScript (ES5.1) ,
HTML5 & CSS3 ,
Vite v8.0.2 ,
Git & GitHub ,
Vercel (Hospedagem).
# 0. ▶️ Como Executar

Siga o passo a passo para rodar o projeto localmente:

# 1. Clone o repositório
git clone https://github.com/Marcosszxleonardo/To-Do-List.git

# 2. Acesse a pasta do projeto
cd To-Do-List

# 3. Instale as dependências
npm install

# 4. Inicialize o projeto
npm run dev
